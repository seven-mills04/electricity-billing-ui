# ⚡ Electricity Billing System

A modern Electricity Billing Management System built using **React**, **Vite**, **Material UI**, and **Axios**, integrated with a **Spring Boot REST API** backend.

## 🚀 Features

### 👥 Consumer Management
- Add new consumers
- Update consumer details
- Delete consumers
- View all registered consumers

### 🔌 Connection Management
- Create electricity connections
- Edit connection details
- Remove connections
- View all connections

### 📊 Meter Reading Management
- Record meter readings
- Update existing readings
- Generate bills from meter readings
- View reading history

### 🧾 Billing Module
- Generate electricity bills
- View bill details
- Calculate bill amount based on consumption

### 💳 Payment Module
- Record bill payments
- Update payment status
- View payment history

### 📈 Dashboard
- Total Consumers
- Total Connections
- Total Bills
- Total Payments
- Statistics Cards
- Charts and Analytics

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/seven-mills04/electricity-billing-ui.git
```

### Navigate to the project

```bash
cd electricity-billing-ui/electricity-billing-ui
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```

---

## 🔗 Backend Configuration

Ensure the Spring Boot backend is running.

Default backend URL:

```
http://localhost:8080
```

Update API URLs inside the `src/api` folder if your backend runs on a different port.
